# Problem

Given a string containing a sequence of digits, generate all possible combinations of numbers that can be formed by using these digits in the same order.

Input: "123"
Expected Output:
- 1, 2, 3
- 12, 3
- 1, 23
- 123

Input: "987654"
Expected Output:
- 9, 8, 7, 6, 5, 4
- 98, 7, 65, 4
- 9, 87, 65, 4
- 987, 6, 54
- 98, 76, 54
- 9876, 54
- 9, 8, 765, 4
- 98, 7654
- 98765, 4
- 9, 876, 54
- 987, 654
- 987654
